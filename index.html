Love at first sight with Ruby on Rails

I took a Programming course at AMA Computer Learning center in my neighborhood but was not able to finish studying due to financial reasons. After that I decided to work in the BPO industry. But even then I still have an urge to learn programming by myself and took a liking with web development in particular.

When I'm off at work I spent a lot of time learning HTML, CSS and JQuery. There are many programming languages and frameworks out there and it is difficult to find what is best or what is right for me. That's when I encountered Ruby on Rails, a web framework that focuses on developer's productivity and happiness as quoted by rubyonrails.org:

Learning to build a modern web application is daunting. Ruby on Rails makes it much easier and more fun. It includes everything you need to build fantastic applications, and you can learn it with the support of our large, friendly community.
